比较

2018-02-23 14:39:41 直通BAT面试算法 算法 0

对于两个32位整数a和b，请设计一个算法返回a和b中较大的。但是不能用任何比较判断。若两数相同，返回任意一个。
给定两个整数a和b，请返回较大的数。
测试样例：

思路：本函数规定符号位：1非负，0负
若a，b符号不同，返回aa符号位+bb符号位，防止溢出
若a，b符号相同，使用a-b的符号位c，即返回ac+b(-c)
