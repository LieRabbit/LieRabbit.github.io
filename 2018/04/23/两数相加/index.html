两数相加（Add Two Numbers）

2018-04-23 12:15:27 LeetCode算法之旅 算法 0

题目描述

给定两个非空链表来表示两个非负整数。位数按照逆序方式存储，它们的每个节点只存储单个数字。将两数相加返回一个新的链表。
你可以假设除了数字 0 之外，这两个数字都不会以零开头。

示例：
输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

解决方法

遍历链表对应相加即可，注意进位

方法一

时间复杂度：O(max(M,N))。M，N分别为l1，l2的链表长度
